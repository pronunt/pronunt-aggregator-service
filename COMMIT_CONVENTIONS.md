# Commit Conventions

Use Conventional Commits for this repository.

Format:

`type(scope): short summary`

Examples:

* `feat(scoring): add risk scoring service`
* `fix(api): return stable PR detail payload`
* `refactor(impact): simplify dependency traversal`
* `docs(readme): update local setup steps`
* `chore(deps): bump FastAPI`
* `test(priority): add prioritization tests`

Recommended types:

* `feat`
* `fix`
* `refactor`
* `docs`
* `chore`
* `test`
* `ci`
* `build`
